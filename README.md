Railway Reservation System

Project Description

This is a Python-based Railway Reservation System that allows users to manage train ticket bookings efficiently through a simple command-line interface.

The system provides functionalities to check seat availability, book tickets, view booking details using a unique booking ID, and cancel tickets. It simulates a basic real-world reservation system using Python data structures like lists and dictionaries.

Features

Check seat availability
Book tickets
View ticket details using booking ID
Cancel tickets
Automatic seat allocation
Unique booking ID generation
Technologies Used

Python
Menu Options

Check Availability
Book Ticket
View Ticket
Cancel Ticket
Exit
How to Run

Make sure Python is installed on your system

Download or clone this repository

Open terminal or command prompt in the project folder

Run the program using:

python main.py
